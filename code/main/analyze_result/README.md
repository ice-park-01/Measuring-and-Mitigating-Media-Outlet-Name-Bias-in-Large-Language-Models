# Analysis Results

This directory stores the output files generated by the analysis scripts in the `analyze/` directory.

## Directory Structure

- **`final/`**: Contains the final aggregated results used for the paper's main claims.
- **`affected_barplot/`**: Bar plots showing how much each media outlet name affects the model's output political leaning.
- **`bias_case_analysis/`**: Detailed logs and data points for specific bias instances.
- **`summarization/`**: Results related to the summarization task experiments.
- **`summarization_rvt/`**: Results from the Relevance Validation Test (RVT) for summaries.

## Key Files

- **`sips_*.csv`**: SIPS (Shift In Poltical Score) metrics for different models and conditions.
- **`robust_id_list_*.txt`**: Lists of sample IDs that were considered "robust" (less biased) or selected for specific analysis sets.
- **`affected_result_df.csv`**: Dataframe containing the quantified effect of media names on model responses.
