# Utility Scripts

Helper scripts for model management and common operations.

## Files Description

- **`download-model.py`**: Script to facilitate downloading of open-source models (e.g., Llama, Gemma, Qwen) from Hugging Face Hub.
- **`tokens.py`**: Utilities for tokenization, including counting tokens and checking context window limits.
- **`functions.py`**: General-purpose helper functions used by other scripts in the codebase.
